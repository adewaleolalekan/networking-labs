In this lab, you'll master the transport layer protocols that power all network communication - including Kubernetes.

## What You'll Learn

- **TCP vs UDP**: Understand when to use each protocol
- **Port Management**: Learn how applications use ports to communicate
- **Service Detection**: Identify what services are running on which ports
- **Network Analysis**: Use tools like netcat, netstat, and ss
- **Kubernetes Application**: See how Services use ports and protocols

## Why This Matters for Kubernetes

Every Kubernetes Service exposes applications on specific ports using TCP or UDP. Understanding these protocols helps you:
- Configure Services correctly
- Troubleshoot connectivity issues
- Secure your applications
- Design efficient microservices
